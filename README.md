# YouTube Song Player

A minimal, elegant YouTube song player with queue functionality.

## Features

- **URL Entry Screen**: Add multiple YouTube video or playlist URLs to a queue
- **Player Screen**: Control playback with play/pause, replay, next, and loop buttons
- **Hidden Player**: Videos play in the background (audio only)
- **Clean Design**: Minimalist interface with a dark theme and fat fun fonts

## Usage

1. Open `index.html` in a web browser
2. Enter YouTube video or playlist URLs
3. Click "Enter another url" to add more URLs to the queue
4. Click "Begin" to start playback
5. Use the control buttons to manage playback

## Color Scheme

- Background: `#121215` (very dark gray)
- Text: `#c5c7f5` (light lavender purple)
- Buttons: `#4d52b8` (blue/purple)

## Technical Details

- Single HTML file with embedded CSS and JavaScript
- Uses YouTube IFrame Player API
- Supports both individual videos and playlists
- Fully responsive design
